# HearthStone

Projekt pri Programiranju 1 z namenom analize igre HearthStone

## Zajeti podatki

Za vsak deck bomo zajeli:

* hero
* type
* craft cost
* mana curve
* cards

Vse podatke bomo zajeli na strani http://www.hearthpwn.com/

## Predvidena analiza

Ogledali si bomo:

* Nadražji deck type
* Povprečen mana curve glede na type
* Cena glede na hero
* TOP 10 Cards
